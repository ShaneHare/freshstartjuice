Production Log

Batch Notes

Every batch is documented — equipment, process, yield, and what we learned. We track everything so every run is better than the last.

Batch 03 Benchmark
Nama J2 Cold Press
Focus
  • Consistent prep, flow, and cleanup
  • Balanced ingredient combinations
  • Continued data tracking across every run
Outcome
  • ~34% yield — best to date
  • Consistent output (~2+ quarts per run)
  • Controlled pulp and minimal waste
Takeaway

Established a repeatable, data-backed process. This batch sets the current performance benchmark.

Batch 02 Transition
Nama J2 Cold Press — Upgrade
Focus
  • Applied process improvements from Batch 01
  • Introduced true cold-press extraction
Outcome
  • Immediate improvement in yield
  • Cleaner juice and better separation
  • Reduced waste and rework
Takeaway

Key inflection point. Combining process discipline with the right equipment significantly improved performance.

Batch 01 Baseline + Foundation
Ninja Professional Blender (Auto IQ)
Focus
  • Began capturing data — ingredient weights, output, yield
  • Built initial prep and cleaning guide
  • Established kitchen workflow and process flow
Outcome
  • Lower yield and higher pulp
  • More manual effort to extract usable juice
Takeaway

Established the foundation. This batch defined the process, exposed limitations, and set the direction for improvement.
